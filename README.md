# YouTube Trending Data Pipeline (India)

A Python-based data engineering project that extracts daily trending YouTube videos for India using the YouTube Data API, processes and aggregates the data, and prepares it for dashboarding.

## Features (Current)
- Extracts daily trending videos for India
- Stores raw data in CSV format
- Clean project structure for scalable pipelines

## Tech Stack
- Python
- YouTube Data API v3
- Pandas
- VS Code
